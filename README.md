## What I work on

I build native iOS apps — SwiftUI, UIKit, Swift. The kind of work that ends up on the App Store and stays there. I've shipped across enterprise staffing platforms, media apps, livestock directories, building management systems, and e-commerce — each with different constraints and different definitions of done.

I own the full cycle: architecture decisions, CI/CD setup, code review standards, App Store submission, Fastlane automation. The job is as much about judgment as it is about writing Swift.

Currently expanding into Flutter for cross-platform work.
